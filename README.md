This repository contains code for jet corrections using modern machine learning techniques.
1. Neural Network approach (neural_network_approach.py): Trains a neural network to separate signal from background using simulated data.
2. Graph Neural Network (GNN) approach: Reduces generator dependence by training the GNN to learn the behavior of the traditional Iterative Constituent Subtraction method, capturing event-by-event fluctuations and achieving improved performance over the standard technique.
